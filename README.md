# 할로우 나이트(Hollow Knight) 모작
DirectX 11을 활용해 할로우 나이트를 모작했습니다.

#### 구현
- 플레이어 (나이트)
- 플레이어 스킬 4종 (공격, 대시, 파이어볼, 집중)
- 몬스터 9종 (보스 : 실패한 기사)
- 맵 (흙의 마을 ~ 잊혀진 교차로 일부)
- 이펙트
- UI
- 타이틀 화면

### 조작
#### 타이틀
- 플레이 시작 : Space Bar (게임 타이틀 등장 이후 동작)


#### 플레이
- 이동 :  →, ←, ↑, ↓
- 점프 : Z (누르는 시간에 따라 점프 높이 조절)
- 대시 : C
- 공격 : X
- 파이어볼 : A 
- 집중 : A (Press)


#### 치트키
- Hp + 1 : B
- Hp - 1 : V
- Mp Max : S
- 다시 시작 : F4
- 프리 카메라 : F8
- 프리 카메라 이동 : W, A, S, D
- 프리 카메라 회전 : 마우스 우클릭 누른 상태로 이동 버튼 입력
